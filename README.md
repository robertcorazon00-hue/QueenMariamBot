## 📦 INSTALLATION

### 1️⃣ Installer Node.js
Téléchargez Node.js v18+ sur https://nodejs.org

### 2️⃣ Installer les dépendances
```bash
npm install
```

### 3️⃣ Démarrer le bot
```bash
node index.js
```

### 4️⃣ Scanner le QR Code
- Un QR code apparaîtra dans le terminal
- Ouvrez WhatsApp → Appareils connectés → Scanner
